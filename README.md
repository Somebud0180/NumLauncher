# NumLauncher
### Launch apps quickly and easily.

Summon apps with just two keys, command and a number. Made for macOS, built with Swift and SwiftUI.

### Feature Roadmap
- Select an app to launch per number. Presenting a file picker restricted to apps (and possibly allow opening other files).
- Configurable head-up notification. Appears at the top-middle of your screen. Similar to Focus and AirPods notifications.
- Select a different modifier key or have a different set of apps per modifier key.
- Disable the shortcuts (specifically 1-4) when Spotlight is open to allow using the quick shortcuts there.
