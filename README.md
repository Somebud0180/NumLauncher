# NumLauncher
### Launch apps quickly and easily.
Summon apps with just two keys, command and a number. Made for macOS, built with Swift and SwiftUI.

<p align=center>
<img width="80%" alt="Screenshots" src="https://github.com/user-attachments/assets/34e7066d-f590-43fd-9597-2b11ff5e7edf" />
</p>

### Features
- Select an app to launch per number.
- A simple toast notification when you launch an app.
- Customize the modifier key. Choose from either the default Command or combine them.
- Disable some shortcuts when Spotlight is open. Allows you to utilize the quick shortcuts.


Made with Swift and SwiftUI. <br>
Supports macOS 14 and above. (Support for versions <26 provided as-is)
